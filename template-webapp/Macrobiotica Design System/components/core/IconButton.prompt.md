Square pill button holding a single icon тАФ toolbar and header actions.

```jsx
<IconButton aria-label="Search"><Icon name="search" /></IconButton>
<IconButton variant="primary" aria-label="Add meal"><Icon name="plus" /></IconButton>
```

- `variant`: `ghost` (default) | `secondary` (outlined) | `primary`
- `size`: `sm` (30px) | `md` (36px) | `lg` (44px)
- Always pass `aria-label`.
